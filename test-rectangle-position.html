🔍 Anna 白板 - 矩形位置测试

测试步骤:

1. 在新标签页打开 http://localhost:8081/
2. 点击 白板 按钮 (id="open-whiteboard")
3. 等待 3 秒让白板加载完成
4. 点击左侧面板的第一个形状按钮 矩形 (class="shape-item", data-tool="rectangle")
5. 在白板中央的浅灰色画布区域点击 (避开左侧面板)
6. 打开浏览器开发者工具 (F12)
7. 在 Console 中查找以 [DBG 开头的日志
8. 将结果粘贴到下面

预期行为分析:

根据代码分析 (src/whiteboard.js 和 core/base/page.js): 默认矩形尺寸: width: 120px height: 80px (定义在 whiteboard.js getDefaultProperties() 函数中) 创建流程: 1. 用户点击画布位置 (x, y) 2. 调用 pageVal.createNew(type, position.x, position.y, undefined, pendingProperties) 3. 矩形的 (x, y) 被设置为点击位置 4. 调用 beginDrag(position.x, position.y) 准备拖拽调整大小 5. 调用 resize(1, 1) 将尺寸临时设为 1x1 6. 如果用户拖拽,尺寸会随鼠标移动而改变 7. 如果用户只点击不拖拽,在 mouseup 时恢复默认尺寸 (120x80) 关键问题: 矩形的 (x, y) 属性表示其左上角位置。 因此: - 如果你在画布 (400, 300) 位置点击 - 矩形的左上角将位于 (400, 300) - 矩形的中心将位于 (400 + 120/2, 300 + 80/2) = (460, 340) 调试日志位置: 查看 Console 中的以下日志: [DBG setTool] want=rectangle, isEmpty=false [DBG onMouseDown] wantedShape not empty, position x=XXX y=YYY [DBG createNew] type=rectangle, x=XXX, y=YYY 其中 x, y 就是矩形左上角的坐标,应该等于你点击的位置。
